# CPU Scheduling Simulator

## Overview

This project is an interactive CPU Scheduling Simulator developed using Python and Streamlit. It allows users to simulate and compare various CPU scheduling algorithms with real-time visualization and performance metrics.

## Features

* Implementation of 8 CPU scheduling algorithms:

  * FCFS (First Come First Serve)
  * SJF (Shortest Job First)
  * LJF (Longest Job First)
  * SRJF (Shortest Remaining Job First)
  * LRJF (Longest Remaining Job First)
  * HRRN (Highest Response Ratio Next)
  * Priority Scheduling (Preemptive)
  * Round Robin

* Interactive user interface using Streamlit

* Dynamic Gantt Chart visualization using Plotly

* Calculation of performance metrics:

  * Waiting Time (WT)
  * Turnaround Time (TAT)
  * Response Time (RT)
  * CPU Utilization
  * Throughput

* Algorithm comparison using tables and graphs

* Analysis of Round Robin with different time quantum values

## Technologies Used

* Python
* Streamlit
* Plotly
* Pandas

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/CPU-Scheduling-Simulator.git
```

2. Navigate to project folder:

```bash
cd CPU-Scheduling-Simulator
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the app:

```bash
streamlit run app.py
```
